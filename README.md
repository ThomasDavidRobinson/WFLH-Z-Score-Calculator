# Offline Z-Score Calculator
This module provides an offline means of calculating z-score.
The tables provided are from the WHO Weight-for-length/height tables as at 11DEC2025
[[https://www.who.int/tools/child-growth-standards/standards/weight-for-length-height]]
___
## Setup
Install the module from the REDCap module repo, enable via control centre, enable for chosen project. 

## Configuration
Z-Score is calculated from height/length, weight, sex, and age. 
These input fields must be selected from the dropdowns, as well as the output zscore field. 
Multiple instances of the fields may be added across the project or per form. Select the correct form per input set to enable it.
